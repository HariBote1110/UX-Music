# 機能詳細 (FEATURES.md)

このドキュメントでは、「UX Music」の主要な機能の目的と詳細な仕様について解説します。

## ✈️ Light Flightモード

### 目的
このモードは、アプリケーションの動作を極限まで軽量化し、CPUやメモリのリソース消費を最小限に抑えることを目的としています。バックグラウンドで音楽を再生しながら他の作業を行う際や、低スペックなPCで利用する際に最適なパフォーマンスを提供します。

### 詳細仕様
Light Flightモードを有効にすると、以下の変更が適用されます。

- **アートワークの非表示**:
  - 曲リストや再生中のアートワークなど、すべてのジャケット画像が非表示になります。
  - ただし、レイアウト崩れを防ぐため、曲リストでは**アルバムアートのグループ表示**が強制的に有効になり、アートワークの代わりにアルバムの区切りを示す罫線が表示されます。
- **ビジュアライザーの変更**:
  - CPU負荷のかかるリアルタイム音声解析ビジュアライザーを停止します。
  - 代わりに、再生中であることを示すシンプルなアニメーションSVGに切り替わります。
- **フレームレートの制限**:
  - アプリ全体の描画更新レート（FPS）を抑えることで、不要なCPU使用率の上昇を防ぎます。

## 🎨 アルバムアートのグループ表示

### 目的
同じアルバムの楽曲がリストに連続して並んでいる場合に、アートワークの表示を最初の1曲のみに集約することで、リストの視認性を向上させ、画面の縦スペースを節約します。特に、1枚のアルバムをじっくり聴く際にリストがすっきりと見やすくなります。

### 詳細仕様
- この機能は、設定画面の「その他」から有効/無効を切り替えられます。
- 有効にすると、同じアルバムの2曲目以降はアートワークが表示されなくなり、代わりにアルバムのまとまりを示すツリー状の罫線が表示されます。
- Light Flightモード有効時は、この機能が設定に関わらず強制的に有効になります。
