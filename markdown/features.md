# 機能一覧 (Features)

UX Music が現在提供している、または計画中の主な機能です。

## 基本的な機能

* **ライブラリ管理**: ローカルの音楽ファイル（mp3, flac, m4a, wav, ogg）をスキャンし、ライブラリとして管理。
* **多彩なビュー**: ライブラリ、アルバム、アーティスト、プレイリスト、最近追加した曲など、様々な表示モード。
* **再生機能**: 標準的な再生、一時停止、次へ、前へ、シーク、音量調整。
* **イコライザー**: 10バンドのグラフィックEQとプリセット機能。
* **リピート・シャッフル**: キュー全体のリピート、1曲リピート、シャッフル再生。
* **再生キュー管理**: 次に再生する曲のリストを管理・編集。
* **プレイリスト**: 自由なプレイリストの作成、曲の追加・削除。
* **アートワーク表示**: 埋め込みアートワークの抽出と表示（`safe-artwork://` プロトコルによる安全な表示）。
* **ラウドネス解析**: EBU R128 に基づくラウドネス値の解析（再生時の音量平準化のため）。
* **歌詞表示/編集**: `.lrc` 同期歌詞の表示に加え、LRCエディタでタイムライン編集（中央移動・左右端リサイズ・ルーラー倍率変更）と既存LRC再編集に対応。
* **TXT自動同期解析 (CoreML)**: LRCエディタから、TXT歌詞をCoreML対応Whisperで自動時刻同期（MLボーカル抽出 `demucs` / カスタム抽出器対応、Apple Silicon では `mps` 優先で抽出し失敗時 `cpu` フォールバック、検知テキスト表示でセグメント確認可能）。
* **Discord RPC**: 現在再生中の曲を Discord のステータスに表示。

## ユニークな機能

* **CDリッピング (UX Rip)**: オーディオCDから高品質に吸い出し、MusicBrainzから情報を自動取得。
* **MTP転送 (UX MTP)**: WalkmanなどのMTPデバイスへmacOSから直接転送。
* **YouTube ダウンロード**: YouTube の動画やプレイリストの URL から直接ライブラリに楽曲を追加。
* **シチュエーション別プレイリスト**: 「朝」「夜」「雨の日」など、曲の解析データに基づき自動生成。
* **再生履歴プレイリスト**: 「よく聴く曲」「最近あまり聴いていない曲」などを自動生成。
* **イントロクイズ**: ライブラリの曲を使ったイントロクイズ機能。
* **ラウドネスノーマライゼーション**: 曲ごとの音量差を平準化（EBU R128準拠）。
* **Light Flightモード**: CPU/メモリ負荷を極限まで抑えた超軽量動作モード。
