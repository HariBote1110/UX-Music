# UX Music Mobile (Legacy)

Archived **Flutter** implementation of the mobile companion. It is **frozen**: use it only if you need the old stack or a reference port of the Wear LAN API client.

Active **iOS** development lives in the sibling directory [`../UX-Music-Mobile`](../UX-Music-Mobile) (Swift + SwiftUI).

## Run (Flutter)

```bash
cd UX-Music-Mobile-Legacy
flutter pub get
flutter run
```

The legacy app uses bundle / application ID suffix `.legacy` so it can be installed alongside the native iOS app.
