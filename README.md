# UX Music

![UX Music Icon](../src/renderer/assets/ux-music-icon.png)

ローカル・オンラインの音源を統合的に管理・再生できる、モダンなデスクトップ音楽プレーヤー。

> [!WARNING]
> **ハイパワーマシン向け**
> このアプリケーションは、楽曲のインポート時に音声解析などの処理を行うため、CPUリソースを大きく消費する場合があります。快適にご利用いただくためには、比較的高性能なPCでの使用を推奨します。

---

## ✨ 主要機能

### 🎵 統合的なライブラリ管理
- **多様なインポート**: ファイルやフォルダのドラッグ＆ドロップ、YouTubeのURL指定（単体・プレイリスト）に対応。
- **アートワーク管理**: アルバム単位でアートワークを管理し、軽量なサムネイルを自動生成することで高速な表示を実現。
- **YouTubeダウンロード**: 「最高品質（映像付き）」か「音声のみ」かを選択可能。ダウンロードした楽曲は自動で音量が正規化されます。

### 💿 高度な再生機能
- **音量ノーマライザー**: 楽曲ごとのラウドネス値（LUFS）を解析し、再生音量を自動で均一化。
- **イコライザー**: サイドバーから手軽に調整できる3バンドEQと、設定画面からの詳細な10バンドグラフィックEQを搭載。
- **多彩な再生モード**: 通常再生、全曲リピート、1曲リピート、シャッフル再生に対応。
- **再生デバイスの切り替え**: アプリ内からオーディオの出力先を動的に変更可能。

### 🎨 洗練されたUI・UX
- **仮想スクロール**: 数千曲規模のライブラリでも軽快なスクロールと高速な起動を実現。
- **Dynamic Island風ビジュアライザー**: 再生中の楽曲のアートワークカラーを反映した、リアルタイムで動くインジケーターを表示。
- **歌詞表示**: 再生中の曲とファイル名が一致する`.lrc`または`.txt`ファイルを自動で検索し、カラオケのように同期表示。
- **Light Flightモード**: アートワークやビジュアライザーを無効化し、CPU・メモリ消費を極限まで抑える軽量モード。

### 🕵️ 隠し機能
- **イントロクイズ**: 設定画面の隠しコマンドからアクセスできるイントロクイズ機能を搭載。再生履歴に基づいた難易度選択や、ハイスコアランキング機能も楽しめます。
